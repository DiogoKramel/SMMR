A probability density function (PDF), or density of a continuous random variable, is a function, whose value at any given sample (or point) in the sample space (the set of possible values taken by the random variable) can be interpreted as providing a relative likelihood that the value of the random variable would equal that sample. The sum of each individual probability should result in 100%.


Cumulative distribution function (CDF) of a variable Y, evaluated at x, is the probability that Y will take a value less than or equal to x. In other words, the plot shows the probability of having the number of strikes in the horizontal axis (or less).
